## 第 1 步：本地压缩 dist

在本地电脑终端中执行：

```bash
# 进入项目目录
cd /Users/xiaokeji/taluo

# 将 dist 目录压缩为 zip 包
zip -r dist.zip dist/
```

执行后会在项目目录下生成 `dist.zip` 文件，用于上传到服务器。

---

## 第 2 步：宝塔新建网站

1. 登录宝塔面板（通常是 `http://服务器IP:8888`）
2. 左侧菜单点击 **网站**
3. 点击顶部 **添加站点**
4. 填写以下信息：

| 配置项 | 填写内容 |
|--------|----------|
| 域名 | `你的域名.com`（如 `tarot.你的域名.com`，或直接用主域名） |
| 根目录 | 默认 `/www/wwwroot/你的域名/`（保持默认） |
| PHP 版本 | **选择「纯静态」**（本项目不需要 PHP） |
| 创建数据库 | **否** |
| 网站分类 | 选填 |

5. 点击 **提交**

---

## 第 3 步：上传并解压

### 3.1 在网站根目录下创建 taluo 目录

进入 `/www/wwwroot/你的域名/` 目录，点击文件夹图标右侧的 **新建目录**，输入 `taluo`，点击确认。

### 3.2 上传 dist.zip

1. 进入 `/www/wwwroot/你的域名/taluo/` 目录
2. 点击顶部工具栏的 **上传** 按钮
3. 选择本地的 `dist.zip` 文件
4. 等待上传完成

### 3.3 解压

1. 在 `/www/wwwroot/你的域名/taluo/` 目录下找到 `dist.zip`
2. 右键点击 → **解压**（或选中后点击顶部 **解压** 按钮）
3. 解压完成后删除 `dist.zip`

### 3.4 确认目录结构

最终目录结构必须为：

```
/www/wwwroot/你的域名/
└── taluo/
    └── dist/
        ├── index.html
        └── assets/
            ├── index-xxx.js
            └── index-xxx.css
```

> ⚠️ 关键检查点：`index.html` 必须位于 `/www/wwwroot/你的域名/taluo/dist/index.html`

## 第 4 步：配置 SSL 证书（必须）

> 本项目使用**摄像头**（WebRTC API），浏览器要求必须在 HTTPS 环境下才能调用摄像头，因此 SSL 证书是**必须配置**的。

1. 在宝塔面板左侧菜单点击 **网站**
2. 找到你刚添加的网站，点击右侧 **设置**
3. 切换到 **SSL** 标签页
4. 选择 **Let's Encrypt**
5. 勾选你的域名
6. 点击 **申请**
7. 申请成功后，找到 **强制 HTTPS** 开关，打开它

> Let's Encrypt 证书免费有效期 90 天，宝塔会自动续期，无需手动管理。

## 常见问题排查

### ❌ 访问后白屏 / 404

| 可能原因 | 解决方法 |
|----------|----------|
| 目录层级不对 | 检查 `dist` 是否在 `taluo` 下，且 `index.html` 在 `dist` 里 |
| 文件权限不足 | 宝塔中右键 `taluo` 目录 → 权限 → 设为 **755** |
| 资源路径不对 | 打开浏览器 F12 → Console 看具体哪个资源 404 |
| 未配 SSL 强制 HTTPS | 访问 `http://` 版本测试，如果正常说明是 HTTPS 问题 |

### ❌ 摄像头无法使用

- 检查是否已配置 SSL 证书（HTTPS）
- 浏览器是否允许摄像头权限（地址栏左侧 🔒 图标 → 网站设置 → 摄像头）
- 建议使用 Chrome 或 Edge 浏览器
- iOS 微信内置浏览器有权限限制，建议用系统 Safari

### ❌ 想要直接通过根域名访问（不要 /taluo/dist/ 路径）

如果你想用 `https://你的域名/` 直接访问：

1. 本地修改 `vite.config.ts`：
   ```ts
   export default defineConfig({
     plugins: [react()],
     base: "/",   // ← 改为 "/"
   });
   ```
2. 重新构建：
   ```bash
   npm run build
   ```
3. 将新的 `dist/` 目录下的**内容**（不是整个 dist 文件夹）直接上传到 `/www/wwwroot/你的域名/`
